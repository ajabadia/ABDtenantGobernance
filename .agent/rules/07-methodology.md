# Methodology (Orchestrator V2)

## 🛠️ Workflow
1. **Brainstorming**: Produce `SPEC.md`.
2. **Implementation Planning**: Produce task plan.
3. **Execution**: TDD + Atomic Commits.
4. **Verification**: Complete test suite + `walkthrough.md`.

## ⚖️ Iron Laws
1. **Secuencia Obligatoria**: No hay código sin Spec y Plan.
2. **Verificación Continua**: Valida cada fase con el usuario.
3. **Trazabilidad**: Todo anclado a la Spec original.
